# Test Plan

## Objective
Evaluate the functional correctness and usability of key user flows in a demo e-commerce web application.

## Scope
### In Scope
- Authentication
- Product navigation
- Cart functionality
- Form validation

### Out of Scope
- Payments
- Performance
- Security
- API testing

## Test Types
- Functional testing
- UI/UX testing
- Negative testing
- Boundary testing

## Environment
- Browser: Chrome
- OS: Windows 10

## Risks
- Cart data loss
- Incorrect validation
- Broken navigation links

## Entry Criteria
- Application accessible

## Exit Criteria
- Planned test cases executed
- All critical defects logged
